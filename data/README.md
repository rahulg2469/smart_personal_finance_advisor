Dataset Instructions
Download the Kaggle Credit Card Transactions Dataset

1. Go to: https://www.kaggle.com/datasets/kartik2112/fraud-detection
2. Sign in to Kaggle (create free account if needed)
3. Click "Download"
4. Unzip the file
5. Place the CSV files in this folder:
   - fraudTrain.csv
   - fraudTest.csv

These files are ignored by git due to size (~490MB). Each team member needs to download them once.
